# Документация для Telegram-бота на Python

## Описание

Этот проект представляет собой Telegram-бота, который использует API OpenRouter для обработки запросов пользователей. Бот может взаимодействовать с пользователями, отвечая на их вопросы и предоставляя информацию.

## Возможности

- **Интерактивные ответы** на сообщения пользователей.
- **Восстановление контекста** общения с командой "Новый запрос".
- Поддержка команд `/start` и `/help`.

## Установка

### Предварительные требования

- Python версии 3.7 или выше.
- Установленные библиотеки: `python-telegram-bot`, `requests`, `python-dotenv`.

## Использование

После запуска бота вы можете начать взаимодействие с ним, нажав кнопку "Старт" или отправив команду /start.

Для получения справки вы можете использовать команду /help.

Чтобы сбросить контекст общения и начать новый запрос, нажмите кнопку "Новый запрос".

## Примеры команд

- /start — Запуск бота и инициализация контекста.
- /help — Получение справочной информации о возможностях бота.
- "Новый запрос" — Сброс контекста общения с ботом.
